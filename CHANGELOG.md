# Changelog

All notable changes to GestureKit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-04

### Added
- Initial release
- Swipe gestures in all four directions
- Double tap and triple tap gestures
- Customizable long press with callbacks
- Pinch/zoom gesture with bounds
- Rotation gesture with snap angles
- Pan gesture with velocity tracking
- Draggable view modifier with bounds and snap-back
- Zoomable view modifier with double-tap zoom
- Rotatable view modifier with snap angles
- Combined transformable modifier (scale + rotate + drag)
- Shake detection for iOS
- Hover effects for macOS and visionOS
- Full test coverage
- Comprehensive documentation

### Platforms
- iOS 15.0+
- macOS 13.0+
- tvOS 15.0+
- watchOS 8.0+
- visionOS 1.0+
